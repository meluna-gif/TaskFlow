TaskFlow - Productivity Dashboard

A full-featured task management web application with:
• Complete task CRUD operations with localStorage persistence
• 4 theme system (Light/Dark/Blue/Black) with user preference saving
• Interactive calendar view for task scheduling
• Progress analytics with Chart.js (completion rates, priority breakdown, weekly trends)
• Data import/export (JSON backup/restore)
• Drag & drop task reordering
• Advanced filtering & sorting (due date, priority, alphabetical)
• Responsive design for all devices
• Mobile-friendly sidebar navigation

Tech Stack: HTML5, CSS3, JavaScript (Vanilla), Chart.js, Font Awesome Icons
